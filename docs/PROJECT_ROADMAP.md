# Project Roadmap: Phase 1 (Android Foundation)

## 🎯 Current Objectives
Goal: Establish a "flawless" native foundation on Android before expanding to Linux/Desktop.

### 📝 Task List
1.  **Task 1: The DAO & Repository Pattern**
    - Define `NodeDao` and `EdgeDao` in Kotlin.
    - Implement Room Database initialization.
2.  **Task 2: The Graph Engine**
    - Port hierarchy/traversal logic from legacy TypeScript to native Kotlin.
3.  **Task 3: Interactive Canvas Prototype**
    - Create a Compose `Canvas` for node rendering.
    - Implement drag-and-drop node interactions.
4.  **Task 4: Haptic Integration**
    - Bind `VibratorManager` to connection events.
5.  **Task 5: Local Inference Bridge**
    - Connect Kotlin service to local Ollama API for node suggestions.
