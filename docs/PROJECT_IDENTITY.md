# Project Identity: OmniMap

## 🌌 Vision
OmniMap is a high-performance, native execution control plane for the human mind. It is a local-first, AI-writable graph of thoughts, tasks, and relationships.

## 📱 Hardware Context
- **Primary Flagship:** Google Pixel 10 Pro (Target for 120Hz smooth animations and tactile polish).
- **Anchor:** CachyOS Desktop (x86_64) for high-bandwidth I/O and local LLM orchestration.

## 🧠 Intelligence Engine
- **Core Model:** `omnimap-architect` (Custom instructions for graph-native project planning).
- **Runtime:** Pivot from local Ollama to **Google Gemini 1.5 Pro API** via official Android SDK for unified, high-bandwidth reasoning while maintaining local-first data integrity.
- **Output:** Structured JSON mutations for real-time graph evolution.

## 💎 Core Values
- **Native over Web:** Absolute disdain for "webby" wrappers. Native components only.
- **Tactile UX:** Every interaction should feel physical (Haptics).
- **Privacy by Design:** 100% Local. No cloud leaks.
