# 📋 Nate's Onboarding Task List: The Path to Development

This list breaks down your environment setup and first mission. Follow these in order.

## Phase 1: Foundation (The Anchor & GitHub)
| Task | Description | Est. Time |
| :--- | :--- | :--- |
| **1.1 Arch Baseline** | Install `git`, `nodejs`, `npm`, `python`, `fish`, and `gh` on the laptop. | 15 mins |
| **1.2 GitHub Identity** | Create an account at github.com and run `gh auth login` on the laptop. | 10 mins |
| **1.3 Gemini Brain** | Install `gemini-cli` on the laptop and configure your API key. | 5 mins |

## Phase 2: Mobile Command (Android Device)
| Task | Description | Est. Time |
| :--- | :--- | :--- |
| **2.1 Termux Init** | Run `pkg update`, `pkg upgrade`, and install the toolchain (node, git, python). | 20 mins |
| **2.2 Bridge Access** | Ensure Shizuku is running and successfully enter the `rish` shell. | 5 mins |
| **2.3 Gemini Brain (Mobile)** | Install `gemini-cli` in Termux and configure your API key. | 10 mins |

## Phase 3: The OmniMap Side-Quest (Choose A or B)
| Task | Description | Est. Time |
| :--- | :--- | :--- |
| **3.1 Repository Entry** | Clone the OmniMap repo into `~/Projects/` on both devices. | 5 mins |
| **3.2 Quest Selection** | Read `NATE_ONBOARDING.md` and tell Gemini: "I choose Quest [A/B/C]". | 2 mins |
| **3.3 Hello World** | Create your first project file (e.g., `omnimap-importer.py` or `index.html`). | 5 mins |
| **3.4 First Commit** | `git add .`, `git commit -m "feat: init nate quest"`, and `git push`. | 5 mins |

### ⏱️ Total Estimated Time: **~1 Hour 20 Mins**
