## ⚡ Your Mind, Visualized.

OmniMap is not a note-taking app. It is a **physics-based neural workspace** designed for those who think in relationships, not lists. Built natively for Android and Desktop, it transforms Google's Gemini 3.1 into a **Proactive Architect** that builds your mind-map alongside you.

---

### 🚀 Core Intelligence & Features

| Feature | Description |
| :--- | :--- |
| **⚛️ Neural Physics** | A high-performance physics engine where nodes have mass, velocity, and spring-based connections. |
| **🤖 Gemini 3.1 Architect** | Proactive AI that analyzes your graph and suggests mutations, projects, and connections. |
| **👻 Drafting Mode** | AI proposals appear as "ghost nodes." Review, commit, or discard mutations with one tap. |
| **🔗 Neural Link** | Real-time P2P synchronization. Turn your Desktop into a Hub and sync your Mobile via local link. |
| **💎 Tactile Flow** | Optimized for 120Hz displays with deep Haptic Snap integration (Android). |
| **🔒 Local-First** | Zero telemetry. Your graph lives in a local Room database. Your keys stay on your device. |

---

### 🎨 The Architect persona
OmniMap doesn't just "chat." The **Architect** persona is designed to generate structured JSON mutations. As you dialogue, it silently drafts new nodes, edges, and project paths directly onto your canvas.

---

### 🛠️ Technical Master-Stack
- **Language:** Kotlin 2.0 (Strictly Idiomatic)
- **UI Framework:** Jetpack Compose / Compose Multiplatform
- **Database:** Room Persistence with FTS4 Search
- **Intelligence:** Google Gemini 3.1 Pro (Official KMP SDK)
- **Sync Architecture:** Lightweight Multi-device Neural Hub (P2P)

### 📥 Getting Started

1.  **Download** the latest APK or Desktop build from the [Releases](https://github.com/DaRipper91/OmniMap/releases) page.
2.  **Initialize:** Open the app and follow the **Intelligence Setup Wizard**.
3.  **Auth:** Log in with Google to instantly link your Gemini 3.1 engine.
4.  **Visualize:** Start creating nodes. Watch the AI Architect propose connections in the background.

### 🏗️ For Developers
```bash
# Clone the repository
git clone https://github.com/DaRipper91/OmniMap.git

# Build the Android APK

# Run the Desktop Application
```
